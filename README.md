Main reference:
>  Li, M.-L., Mauro, F. D., Candan, K. S., & Sapino, M. L. (2021). Matrix Factorization with Interval-Valued Data. In IEEE Transactions on Knowledge and Data Engineering (Vol. 33, Issue 4, pp. 1644â€“1658). Institute of Electrical and Electronics Engineers (IEEE). https://doi.org/10.1109/tkde.2019.2942310

* ISVD 2
* ISVD 3
* ISVD 4
* Supporting tools
